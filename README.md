# Rock-Paper-Scissors
## What is This?

This is a console based rock paper scissors game

## How To Play

- Open index.html
- Open Console by pressing F12 and choose "Console" tab
- Enter the word "rock" or "paper" or "scissors" into prompt
- The prompt will retry for your answer if you typed incorrectly
- The entry is case insensitive
- The result of the game will be written in console
